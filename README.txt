client.py
	라즈베리파이 (GPS 송수신기) 의 클라이언트 코드
config.txt
	라즈베리파이 (GPS 송수신기) 의 설정 값
	줄 순서대로
		클라이언트 버전
		타입
		이름
		서버 주소
	로 설정됨. 앞의 3개는 데이터베이스에 들어갈 항목이며 서버 주소는 현재 포팅된 서버 주소를 입력.
	ngrok 을 사용해서 서버를 외부로 포팅할 경우 ngrok 껐다 킬때마다 서버 주소가 바뀌니 클라이언트 설정 파일도 바꿔줘야함.

start_servers.sh
	실행 시 맵 서버와 웹 서버 자동으로 실행

./project/frontend-react/src/config.js
	현재 서버 주소 값 설정.
	역시 ngrok 을 사용하면 껐다 킬때마다 바뀌므로 이 값을 동기화 해줘야 함.
	자재 관리 시스템의 프론트엔드에서 서버를 거쳐야하는 로직은 모두 이 config.js 의 서버 주소값을 참고하여 실행되므로 지도가 안뜨거나 데이터베이스가 안뜨면 이 값을 참고
	
	
실행 방법
	start_servers.sh 를 실행하면 localhost:8080 또는 외부 개방한 서버 주소에 자재 관리 서비스가 동작합니다.
	단 mysql 서버를 사용해서 로컬에 mysql 서버가 동작하고 있어야 하며 아이디/비밀번호는 root/root 로 기본 설정되어 있습니다.
	변경하고 싶다면 ./project/server.js 코드를 수정하면 됩니다.
	처음 실행 시 사용하는 데이터베이스는 자동으로 구축됩니다.
	
클라이언트 LTE 사용 관련
	sixfab LTE 모듈을 사용하는데 USIM 카드 잔액이 현재 18달러 남아있습니다. 계속 켜두면 돈 나가서 일단은 꺼놓고 나중에 시연 등 실제 LTE 통신을 사용할 일이 있으면 연락 주세요. LTE 껐다 켰다하는 기능이 sixfab 관리 웹에 제 구글 계정으로 있어서 연락 주시면 들어가서 키겠습니다.
